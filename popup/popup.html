How to use UUID Resolver

  1. 📋 Select text containing a UUID on any Workspace ONE UEM page
  2. 🖱️ Right-click and select "Resolve UUID"
  3. ✅ The extension will automatically identify and display the entity details

Supported entity types (in order of resolution):

  • 🏷️ Tags
  • 📜 Scripts/Workflows
  • 🏢 Organization Groups
  • 📱 Applications
  • 📦 Products
  • ⚙️ Profiles
