üîß Troubleshooting

  • Context menu not appearing? Make sure you have selected some text first, then right-click
  • No "Resolve UUID" option? Check that the extension is loaded and enabled in chrome://extensions/
  • Nothing happens after clicking "Resolve UUID"? Open Chrome DevTools (F12) and check console for errors
  • No notification appears? Check if Chrome notifications are enabled for the extension
  • Settings not saving? Check the browser console for error messages
  • Connection test failing? Verify your UEM server URL and credentials
